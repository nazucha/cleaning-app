# @google/genai インストール - 代替方法

## 現在の状況

npmキャッシュの権限エラーが発生しています。以下の方法を試してください。

## 方法1: Macの管理者パスワードを使用（推奨）

ターミナルで以下を実行してください。Macのログインパスワード（管理者パスワード）の入力が求められます：

```bash
sudo chown -R $(whoami) ~/.npm
```

**注意**: パスワードを入力する際、画面には何も表示されませんが、入力はされています。入力後、Enterキーを押してください。

その後、以下を実行：

```bash
cd /Users/malove/Desktop/再チャレンジ/cleaning-app
npm install @google/genai
```

## 方法2: npmキャッシュを無効化してインストール

```bash
cd /Users/malove/Desktop/再チャレンジ/cleaning-app

# 一時的にキャッシュを無効化
npm install @google/genai --cache /tmp/npm-cache-$(whoami)
```

## 方法3: プロジェクトローカルにnode_modulesを作成

```bash
cd /Users/malove/Desktop/再チャレンジ/cleaning-app

# プロジェクト内にキャッシュディレクトリを作成
mkdir -p .npm-cache

# キャッシュディレクトリを指定してインストール
npm install @google/genai --cache .npm-cache
```

## 方法4: yarnを使用（npmの代替）

yarnがインストールされている場合：

```bash
cd /Users/malove/Desktop/再チャレンジ/cleaning-app
yarn add @google/genai
```

## インストール確認

どの方法でも、インストール後は以下で確認：

```bash
npm list @google/genai
# または
ls node_modules/@google/genai
```

## 推奨

**方法1（sudoコマンド）**が最も確実です。Macのログインパスワードを入力すれば解決します。

もしパスワードがわからない場合は、Macのシステム設定でパスワードをリセットするか、システム管理者に相談してください。
