# 完全な型番データ取得手順

## 概要

https://iair-c.com/manual/index/ から各型番の詳細ページをスクレイピングして、完全な型番（例：「AJT40SEP」）と取扱説明書からお掃除機能の情報を取得します。

## スクリプトの実行

### 1. ダイキンの完全な型番データを取得

```bash
cd /Users/malove/Desktop/再チャレンジ/cleaning-app
node scripts/scrapeDaikinFullModels.js
```

このスクリプトは以下を実行します：
1. `daikinModelDetails.json`から既存の型番（例：「AJT-SEP」）を読み込む
2. 各型番の詳細ページ（例：https://iair-c.com/manual/daikin/daikin-a/ajt-sep/）にアクセス
3. ページから完全な型番（例：「AJT40SEP」）を抽出
4. 取扱説明書からお掃除機能の情報を抽出
5. 結果を`daikinFullModelDetails.json`に保存

## 出力ファイル

### `data/daikinFullModelDetails.json`
完全な型番データが保存されます：
```json
{
  "manufacturer": "ダイキン",
  "source": "https://iair-c.com/manual/daikin/index-daikin/",
  "lastUpdated": "2026-01-26",
  "totalFullModels": 1234,
  "modelDetails": {
    "AJT": [
      "AJT40SEP",
      "AJT22SEP",
      "AJT40SES",
      ...
    ],
    ...
  }
}
```

### `data/daikinCleaningFunctionFromManual.json`
取扱説明書から取得したお掃除機能の情報が保存されます：
```json
{
  "AJT-SEP": {
    "AJT40SEP": true,
    "AJT22SEP": true,
    ...
  },
  ...
}
```

## 注意事項

- スクリプトの実行には時間がかかります（各型番のページにアクセスするため）
- サーバーに負荷をかけないように、各リクエストの間に500msの待機時間を設けています
- すべての型番を処理するには、数時間かかる可能性があります

## 次のステップ

1. スクリプトを実行して完全な型番データを取得
2. ブラウザをリロードして、候補リストに完全な型番が表示されることを確認
3. 必要に応じて、他のメーカー（三菱電機、パナソニックなど）にも同様のスクリプトを作成
