# Web版トラブルシューティング

## 問題: ブラウザで空白ページが表示される

### 確認事項

1. **ブラウザのコンソールを確認**
   - F12 または Cmd + Option + I（Mac）で開発者ツールを開く
   - Consoleタブでエラーメッセージを確認
   - エラーメッセージを共有してください

2. **ネットワークタブを確認**
   - 開発者ツールのNetworkタブを開く
   - リソースが正しく読み込まれているか確認
   - エラー（赤色）が表示されていないか確認

3. **キャッシュをクリア**
   - ブラウザのキャッシュをクリア（Cmd + Shift + R）
   - または、シークレットモードで開く

### よくある原因

1. **JavaScriptエラー**
   - ブラウザのコンソールにエラーが表示されている
   - エラーメッセージを確認して修正

2. **リソースの読み込みエラー**
   - ネットワークタブでエラーを確認
   - 必要なファイルが読み込まれていない

3. **React Native Webの設定問題**
   - Web版用のパッケージが正しくインストールされていない
   - 設定ファイルに問題がある

### 次のステップ

ブラウザのコンソールに表示されているエラーメッセージを共有してください。
それに基づいて修正を行います。
