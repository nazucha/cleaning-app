## ステップ4: 環境変数の設定

### .envファイルの編集

プロジェクトのルートディレクトリ（`/Users/malove/Desktop/再チャレンジ/cleaning-app`）にある`.env`ファイルを編集します。

### 設定内容

`.env`ファイルに以下を追加または更新：

```env
# Vertex AIを使用
USE_VERTEX_AI=true
GOOGLE_CLOUD_PROJECT=ambient-sylph-485519-b4

# Developer APIの設定は削除またはコメントアウト
# GEMINI_API_KEY=...
```

### .envファイルが存在しない場合

`.env`ファイルが存在しない場合は、新規作成してください：

```bash
cd /Users/malove/Desktop/再チャレンジ/cleaning-app
touch .env
```

その後、上記の内容を追加してください。

### 設定確認

`.env`ファイルに以下が設定されているか確認：

- ✅ `USE_VERTEX_AI=true`
- ✅ `GOOGLE_CLOUD_PROJECT=ambient-sylph-485519-b4`
