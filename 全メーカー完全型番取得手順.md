# 全メーカーの完全な型番データ取得手順

## 概要

https://iair-c.com/manual/index/ から各メーカーの型番の詳細ページをスクレイピングして、完全な型番（例：「AJT40SEP」）と取扱説明書からお掃除機能の情報を取得します。

## スクリプトの実行

### 個別に実行する場合

各メーカーごとに個別に実行できます：

```bash
cd /Users/malove/Desktop/再チャレンジ/cleaning-app

# ダイキン
node scripts/scrapeDaikinFullModels.js

# 三菱電機
node scripts/scrapeMitsubishiFullModels.js

# パナソニック
node scripts/scrapePanasonicFullModels.js

# 日立
node scripts/scrapeHitachiFullModels.js

# 東芝
node scripts/scrapeToshibaFullModels.js

# シャープ
node scripts/scrapeSharpFullModels.js

# 三菱重工
node scripts/scrapeMitsubishiHeavyFullModels.js

# 富士通ゼネラル
node scripts/scrapeFujitsuFullModels.js
```

### 一括実行する場合

`scrapeAllManufacturersFullModels.js`を更新して、全メーカーを一括実行することもできます（現在は一部のメーカーのみ対応）。

## 出力ファイル

各メーカーごとに以下のファイルが作成されます：

- `data/{メーカー名}FullModelDetails.json` - 完全な型番データ
- `data/{メーカー名}CleaningFunctionFromManual.json` - 取扱説明書から取得したお掃除機能の情報

例：
- `data/daikinFullModelDetails.json`
- `data/mitsubishiFullModelDetails.json`
- `data/panasonicFullModelDetails.json`
- など

## 注意事項

- スクリプトの実行には時間がかかります（各型番のページにアクセスするため）
- サーバーに負荷をかけないように、各リクエストの間に500msの待機時間を設けています
- すべての型番を処理するには、各メーカーで数時間かかる可能性があります

## 次のステップ

1. 各メーカーのスクリプトを実行して完全な型番データを取得
2. ブラウザをリロードして、候補リストに完全な型番が表示されることを確認
3. `App.tsx`は既に更新済みで、各メーカーの完全な型番データを自動的に読み込みます
