# タイムアウトエラーの対処法

## エラー: "The request timed out"

このエラーは通常、以下の原因で発生します：

### 解決方法1: Expoサーバーを再起動

1. 現在のExpoサーバーを停止（Ctrl + C）
2. キャッシュをクリアして再起動：
   ```bash
   npx expo start --clear
   ```

### 解決方法2: ポートを変更

別のポートで起動：
```bash
npx expo start --port 8083
```

### 解決方法3: 通常モードで起動

LANモードではなく、通常モードで起動：
```bash
npm start
```

### 解決方法4: ネットワーク接続を確認

- PCとスマホが同じWi-Fiネットワークに接続されているか確認
- ファイアウォールの設定を確認
- VPNを無効化してみる

### 解決方法5: 手動でURLを入力

1. ターミナルに表示されているURLを確認
2. Expo Goアプリで「Enter URL manually」を選択
3. URLを入力（例: `exp://192.168.128.186:8082`）

### 解決方法6: 開発ビルドを使用

Expo Goではなく、開発ビルドを使用する：
```bash
npx expo run:ios
# または
npx expo run:android
```
