## 次のステップ

### ブラウザで確認すること

1. **ページを開く**: `http://localhost:8081`
2. **ページをリロード**: Cmd + Shift + R（Mac）または Ctrl + Shift + R（Windows）
3. **コンソールを開く**: Cmd + Option + I（Mac）または F12（Windows）
   - Consoleタブで以下を確認：
     - "=== HTML loaded ===" が表示されるか
     - "=== Window loaded ===" が表示されるか
     - "Root element:" が表示されるか
     - エラーメッセージ（赤色）がないか

4. **ネットワークタブを確認**:
   - `AppEntry.bundle`のステータスが200か確認
   - ファイルサイズが表示されているか確認

## 問題が続く場合

以下の情報を共有してください：
1. コンソールに表示されているメッセージ（すべて）
