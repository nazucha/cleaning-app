# 簡単な確認方法

## ステップ1: ブラウザでページを開く

1. ブラウザ（Chrome、Safari、Firefoxなど）を開く
2. アドレスバーに `http://localhost:8081` と入力してEnterキーを押す

## ステップ2: ページのソースを確認

### Macの場合
- **Cmd + Option + U** を押す
- または、ページ上で右クリック → 「ページのソースを表示」

### Windowsの場合
- **Ctrl + U** を押す
- または、ページ上で右クリック → 「ページのソースを表示」

### 確認すること
- `<div id="root"></div>` が存在するか
- `<script src="...AppEntry.bundle?platform=web..."></script>` が存在するか

## ステップ3: コンソールを開く（簡単な方法）

### Macの場合
- **Cmd + Option + I** を押す（Chrome/Edge）
- または、**Cmd + Option + C** を押す（Safari）

### Windowsの場合
- **F12** を押す
- または、**Ctrl + Shift + I** を押す

## ステップ4: コンソールで確認すること

1. **Consoleタブ**をクリック
2. 以下のメッセージが表示されるか確認：
   - "=== index.js: Web版で実行中 ==="
   - "registerRootComponentを呼び出します..."
   - "App component rendering..."
   - "=== App Component Debug ==="

3. **エラーメッセージ（赤色）**がないか確認

## ステップ5: ページに何か表示されているか確認

- ページが完全に空白か
- 「テスト: このテキストが表示されれば、アプリは動作しています」というテキストが表示されるか

## 問題が続く場合

以下の情報を共有してください：
1. ページのソースに `<div id="root"></div>` が存在するか
2. ページのソースに `<script src="...AppEntry.bundle?platform=web..."></script>` が存在するか
3. コンソールに表示されているメッセージ（エラーがあればその内容）
4. ページに何か表示されているか（完全に空白か、何か表示されているか）
