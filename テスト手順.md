# テスト手順

## 現在の状況
- HTMLには `<div id="root"></div>` が存在します
- HTMLには `<script src="/index.bundle?platform=web..."></script>` が存在します
- バンドルは生成されています

## 次の確認ステップ

### 1. ブラウザでページを開く
- `http://localhost:8081` を開く

### 2. ページのソースを確認（Cmd + Option + U または Ctrl + U）
以下の内容が含まれているか確認：
```html
<div id="root"></div>
<script src="/index.bundle?platform=web&dev=true&hot=false&lazy=true&transform.engine=hermes&transform.routerRoot=app&unstable_transformProfile=hermes-stable" defer></script>
```

### 3. コンソールを開く（Cmd + Option + I または F12）
以下のメッセージが表示されるか確認：
- "=== index.js: Web版で実行中 ==="
- "registerRootComponentを呼び出します..."
- "App component rendering..."

### 4. ページに表示される内容を確認
以下のいずれかが表示されるか確認：
- 「テスト: このテキストが表示されれば、アプリは動作しています」というテキスト
- 「清掃サービス問い合わせ」というタイトル
- 完全に空白

### 5. ネットワークタブを確認
1. コンソールの「Network」タブを開く
2. ページをリロード（Cmd + R または F5）
3. `index.bundle?platform=web...` というリクエストが存在するか確認
4. そのリクエストのステータスが200（成功）か確認

## 問題が続く場合

以下の情報を共有してください：
1. ページのソースに上記の内容が含まれているか
2. コンソールに表示されているメッセージ（すべて）
3. ネットワークタブで `index.bundle` のステータス
4. ページに何か表示されているか
