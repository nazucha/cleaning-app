# Vertex AI 認証設定手順

## プロジェクトID
`ambient-sylph-485519-b4`

## ステップ3: 認証の設定

### 方法A: gcloud CLIを使用（推奨）

#### 3-1. gcloud CLIのインストール確認

ターミナルで以下を実行して、gcloud CLIがインストールされているか確認：

```bash
which gcloud
```

#### 3-2. gcloud CLIがインストールされていない場合

1. 以下のコマンドでインストール：
   ```bash
   # macOSの場合
   brew install --cask google-cloud-sdk
   ```

2. または、公式サイトからインストール：
   - https://cloud.google.com/sdk/docs/install

#### 3-3. 認証の実行

ターミナルで以下を実行：

```bash
# 認証（ブラウザが開きます）
gcloud auth application-default login

# プロジェクトを設定
gcloud config set project ambient-sylph-485519-b4
```

**注意**: `gcloud auth application-default login`を実行すると、ブラウザが開いてGoogleアカウントでログインを求められます。ログイン後、認証が完了します。

### 方法B: サービスアカウントキーを使用（gcloud CLIが使えない場合）

1. Google Cloud Consoleで「IAMと管理」→「サービスアカウント」を開く
2. 「サービスアカウントを作成」をクリック
3. サービスアカウント名を入力（例: `vertex-ai-user`）
4. 「作成して続行」をクリック
5. 「ロール」で「Vertex AI User」を選択
6. 「続行」→「完了」をクリック
7. 作成したサービスアカウントをクリック
8. 「キー」タブ→「キーを追加」→「新しいキーを作成」
9. 「JSON」を選択→「作成」
10. ダウンロードしたJSONファイルのパスをメモ

## 次のステップ

認証が完了したら、ステップ4（環境変数の設定）に進みます。
