# PlatformConstantsエラーの対処法

## 問題
`PlatformConstants`が見つからないというエラーが発生しています。これはSDK 54の新しいアーキテクチャ（Bridgeless mode）に関連する問題です。

## 解決方法

### 方法1: Web版で動作確認（推奨）
Webブラウザでアプリを開いて動作確認：
```bash
npx expo start --web
```

### 方法2: 開発者メニューで「Reload JS」
1. アプリ画面を3本指で長押し
2. 開発者メニューが開く
3. 「Reload」をタップ

### 方法3: Expo Goアプリを再インストール
1. Expo Goアプリを削除
2. App Storeから最新版を再インストール
3. 再度QRコードをスキャン

### 方法4: より基本的なExpoテンプレートから始める
新しいプロジェクトを作成して、段階的に機能を追加

### 方法5: 開発ビルドを使用
Expo Goではなく、開発ビルドを使用：
```bash
npx expo run:ios
```

## 現在の状況
- SDK 54.0.0を使用
- Expo GoアプリはSDK 54をサポート
- しかし、新しいアーキテクチャの互換性問題が発生

## 推奨
まずはWeb版で動作確認し、フォームが正常に動作することを確認してください。
