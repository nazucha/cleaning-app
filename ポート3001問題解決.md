# ポート3001問題解決

## 問題

ポート3001が既に使用中（プロセスID: 53159）のため、サーバーが起動できない。

## 解決方法

### 方法1: 別のポートを使用（推奨・実施済み）

ポート3002に変更しました：

1. **`.env`ファイル**: `PORT=3002`を追加
2. **`App.tsx`**: ポート3002に変更

サーバーを起動：

```bash
cd /Users/malove/Desktop/再チャレンジ/cleaning-app
node server.js
```

サーバーはポート3002で起動します。

### 方法2: 既存のプロセスを終了

ターミナルで以下を実行：

```bash
# プロセスを終了
kill 53159

# または強制終了
kill -9 53159

# その後、サーバーを起動
cd /Users/malove/Desktop/再チャレンジ/cleaning-app
node server.js
```

## 確認

サーバーが起動したら、以下のログが表示されます：

```
サーバーがポート3002で起動しました
```

ブラウザでアプリを開き、型番を入力してAIによる概要が表示されるか確認してください。
