# 開発ビルドの作成手順（簡易版）

## 前提条件
- Expoアカウントが必要です（無料で作成できます）

## 手順

### 1. Expoアカウントにログイン
```bash
npx eas-cli login
```

### 2. プロジェクトを設定
```bash
npx eas-cli build:configure
```

### 3. 開発ビルドを作成
```bash
# iOSシミュレーター用
npx eas-cli build --profile development --platform ios

# 実機用（ローカルビルド、Xcode必要）
npx eas-cli build --profile development --platform ios --local
```

## 注意事項
- クラウドビルドは初回に時間がかかります（10-20分程度）
- ビルドが完了すると、ダウンロードリンクが表示されます
- ダウンロードしたIPAファイルを実機にインストールできます

## 代替方法
Xcodeがインストールされている場合は、ローカルでビルドすることもできます：
```bash
npx expo run:ios
```
