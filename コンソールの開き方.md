# ブラウザのコンソールの開き方

## Macの場合

### Google Chrome / Microsoft Edge
1. **Cmd + Option + I** を押す
2. または、メニューから「表示」→「開発/管理」→「デベロッパーツール」

### Safari
1. まず、Safariの環境設定で「開発」メニューを有効にする必要があります
2. Safari → 環境設定 → 詳細 → 「メニューバーに"開発"メニューを表示」にチェック
3. その後、**Cmd + Option + C** を押す

## Windowsの場合

### Google Chrome / Microsoft Edge
1. **F12** を押す
2. または、**Ctrl + Shift + I** を押す

## コンソールで確認すること

1. **Consoleタブ**を開く
2. エラーメッセージ（赤色）がないか確認
3. ログメッセージ（"App component rendered"など）が表示されるか確認
