## 🔍 確認すべき項目

### 1. ブラウザでアプリが表示されるか
- [ ] ブラウザで `http://localhost:8081` を開く
- [ ] 「清掃サービス問い合わせ」というタイトルが表示される
- [ ] フォームが表示される（空白ページではない）

### 2. ブラウザのコンソールでエラーがないか
- [ ] F12 または Cmd + Option + I で開発者ツールを開く
- [ ] Consoleタブでエラーメッセージ（赤色）がないか確認
- [ ] "App component rendered" というログが表示されるか確認

### 3. フォームが動作するか
- [ ] テキスト入力フィールドに文字を入力できる
- [ ] エアコン情報を追加できる
- [ ] 送信ボタンが表示される

### 4. バックエンドAPIが動作するか
- [ ] `http://localhost:3000/api/health` にアクセスして `{"status":"ok"}` が返る
- [ ] フォーム送信時にバックエンドAPIにリクエストが送信される

## 🐛 問題が発生した場合

### 空白ページが表示される場合
1. ブラウザのコンソールでエラーメッセージを確認
2. エラーメッセージを共有してください

### フォームが表示されない場合
1. ブラウザをリロード（Cmd + Shift + R）
2. キャッシュをクリアして再度開く

### 送信ボタンを押しても動作しない場合
1. バックエンドサーバーが実行中か確認
2. ブラウザのコンソールでエラーメッセージを確認

## 📝 次のステップ

アプリが正常に表示されたら：
1. フォームにデータを入力してテスト
2. 送信ボタンを押してGoogle Sheetsにデータが送信されるか確認
