### 2. Expo Webサーバー
- **状態**: ✅ 起動中（バックグラウンド）
- **URL**: http://localhost:8081

## 📱 アプリの使用方法

### Web版で開く
1. ブラウザで以下のURLを開く:
   ```
   http://localhost:8081
   ```

2. 「清掃サービス問い合わせ」フォームが表示されます

3. フォームに入力して送信ボタンを押すと、Google Sheetsにデータが送信されます

## 🔧 トラブルシューティング

### アプリが表示されない場合
- ターミナルで `npx expo start --web` を実行
- ブラウザで `http://localhost:8081` を開く

### 送信が失敗する場合
- バックエンドサーバーが起動しているか確認: `curl http://localhost:3000/api/health`
- サーバーが起動していない場合: `npm run server` を実行
