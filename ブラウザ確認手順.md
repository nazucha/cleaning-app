# ブラウザ確認手順

## 現在の状況
- ページが空白で表示される
- エラーメッセージも表示されない

## 確認手順

### 1. ブラウザでページを開く
- `http://localhost:8081` を開く

### 2. ページのソースを確認（Cmd + Option + U または Ctrl + U）
以下の内容が含まれているか確認：
- `<div id="root">` が存在するか
- `<script src="/node_modules/expo/AppEntry.bundle?platform=web..."></script>` が存在するか

### 3. コンソールを開く（Cmd + Option + I または F12）
**Consoleタブ**で以下を確認：
- "=== HTML script loaded ===" というメッセージが表示されるか
- "=== Window loaded ===" というメッセージが表示されるか
- **エラーメッセージ（赤色）**がないか

### 4. ネットワークタブを確認
1. コンソールの**Networkタブ**を開く
2. ページをリロード（Cmd + R または F5）
3. `AppEntry.bundle?platform=web...` というリクエストを探す
4. そのリクエストの**ステータス**を確認：
   - **200**: 成功（正常に読み込まれている）
   - **404**: ファイルが見つからない
   - **500**: サーバーエラー
   - **その他**: エラー

### 5. ページに何か表示されているか確認
- 「バンドルを読み込んでいます...」というテキストが表示されるか
- それとも完全に空白か

## 次のステップ

以下の情報を共有してください：
1. **コンソール**: 表示されているメッセージ（すべて）
2. **ネットワークタブ**: `AppEntry.bundle`のステータスコード（200、404、500など）
3. **ページ**: 何か表示されているか（完全に空白か、テキストが表示されているか）

これらの情報があれば、問題を特定できます。
