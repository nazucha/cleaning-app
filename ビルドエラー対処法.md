# ビルドエラーの対処法

## エラー内容
iOSビルドが失敗しました。Prebuildフェーズでエラーが発生しています。

## 確認事項

### 1. ログを確認
以下のURLでログを確認してください：
https://expo.dev/accounts/nazuasa/projects/cleaning-app/builds/906ac5e3-5f8d-4f53-97c4-2a9077a30f94

### 2. よくある原因と解決方法

#### アセットファイルの不足
- `assets/icon.png` が存在しない
- `assets/splash.png` が存在しない
- `assets/adaptive-icon.png` が存在しない

**解決方法:**
```bash
# アセットディレクトリを作成
mkdir -p assets

# 一時的なアイコンを作成（後で本物のアイコンに置き換える）
# または、app.jsonからアセットの参照を削除
```

#### app.jsonの設定問題
- バンドルIDが正しく設定されていない
- 必要な設定が不足している

**解決方法:**
app.jsonを確認して、必要な設定を追加

#### 依存関係の問題
- パッケージのバージョンが互換性がない

**解決方法:**
```bash
npm install
```

## 次のステップ
1. ログを確認して具体的なエラー内容を把握
2. アセットファイルを作成またはapp.jsonから削除
3. 再度ビルドを実行
