# @google/genai パッケージインストール手順（権限エラー対応）

## 現在の状況

npmのキャッシュフォルダの権限エラーが発生しています。以下の手順で解決してください。

## 解決方法

### 方法1: npmキャッシュの権限を修正（推奨）

ターミナルで以下を実行してください：

```bash
# npmキャッシュの権限を修正
sudo chown -R $(whoami) ~/.npm

# パッケージをインストール
cd /Users/malove/Desktop/再チャレンジ/cleaning-app
npm install @google/genai
```

### 方法2: npmキャッシュをクリアして再試行

```bash
# npmキャッシュをクリア
npm cache clean --force

# パッケージをインストール
cd /Users/malove/Desktop/再チャレンジ/cleaning-app
npm install @google/genai
```

### 方法3: package.jsonに追加済みの場合

`package.json`には既に`@google/genai`を追加済みです。以下のコマンドでインストールを試してください：

```bash
cd /Users/malove/Desktop/再チャレンジ/cleaning-app
npm install
```

## インストール確認

インストールが完了したら、以下で確認してください：

```bash
npm list @google/genai
```

`@google/genai@x.x.x`と表示されれば成功です。
