# リアルタイム進捗確認方法

## 方法1: 最新のログファイルをリアルタイムで確認（推奨）

```bash
cd /Users/malove/Desktop/再チャレンジ/cleaning-app

# 最新のログファイルを特定して監視
tail -f logs/all_manufacturers_*.log | grep -v "^#"
```

または、最新のログファイルを直接指定：

```bash
# 最新のログファイルを確認
LATEST_LOG=$(ls -t logs/all_manufacturers_*.log 2>/dev/null | head -1)

# リアルタイムで監視
tail -f "$LATEST_LOG"
```

## 方法2: 個別のメーカーログをリアルタイムで確認

現在処理中のメーカーのログを直接監視：

```bash
cd /Users/malove/Desktop/再チャレンジ/cleaning-app

# 三菱電機（現在処理中）
tail -f logs/scrapeMitsubishiFullModels.log

# パナソニック（次に処理される）
tail -f logs/scrapePanasonicFullModels.log

# 日立
tail -f logs/scrapeHitachiFullModels.log

# 東芝
tail -f logs/scrapeToshibaFullModels.log

# シャープ
tail -f logs/scrapeSharpFullModels.log

# 三菱重工
tail -f logs/scrapeMitsubishiHeavyFullModels.log

# 富士通ゼネラル
tail -f logs/scrapeFujitsuFullModels.log
```

## 方法3: すべてのログを同時に監視

```bash
cd /Users/malove/Desktop/再チャレンジ/cleaning-app

# すべてのログファイルを同時に監視
tail -f logs/*.log
```

## 方法4: スクリプトを直接実行してリアルタイム表示（推奨）

バックグラウンド実行ではなく、直接実行すると画像のようなリアルタイム表示が見られます：

```bash
cd /Users/malove/Desktop/再チャレンジ/cleaning-app

# 三菱電機を直接実行（リアルタイム表示）
node scripts/scrapeMitsubishiFullModels.js

# パナソニックを直接実行（リアルタイム表示）
node scripts/scrapePanasonicFullModels.js

# 日立を直接実行（リアルタイム表示）
node scripts/scrapeHitachiFullModels.js
```

## 方法5: 進捗を定期的に確認

```bash
cd /Users/malove/Desktop/再チャレンジ/cleaning-app

# 5秒ごとに最新のログを表示
watch -n 5 'tail -20 logs/all_manufacturers_*.log 2>/dev/null | tail -20'
```

## 現在の実行状況を確認

```bash
cd /Users/malove/Desktop/再チャレンジ/cleaning-app

# 最新のログファイルの最後の30行を表示
tail -30 logs/all_manufacturers_*.log 2>/dev/null | tail -30

# または、最新のログファイルを特定
LATEST=$(ls -t logs/all_manufacturers_*.log 2>/dev/null | head -1)
tail -30 "$LATEST"
```

## 注意事項

- `tail -f`を停止するには `Ctrl+C` を押してください
- 複数のターミナルウィンドウを開いて、異なるログを同時に監視することもできます
- 直接実行すると、画像のようなリアルタイム表示が確認できますが、ターミナルを閉じると処理が中断されます
